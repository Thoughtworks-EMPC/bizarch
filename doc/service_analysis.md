# 数据分析服务 
## 定位    
处理跨领域多维度的数据分析

## 业务能力  
* 资产负债分析
* 风险测算
* 成本管控
* 利润分析
* 行内报送

### 资产负债分析  
同业资产规模分析表（总部纬度/分部纬度），资产剩余现状、资产原始期限现状，现有资产跨季分析表（每日动态变化，报表加文字）、未来1个月、3个月，6个月及五年内各年内到期情况（每日动态变化，报表加文字）、每月出账汇总各类资产，对负债组合的结构测算和预测

### 成本管控
针对存单和活期的成本进行检视。存单的成本管控方面需要产出存单发行的对标利率与金额

### 利润分析
对已发生的资产负债业务进行盈利分析

### 行内报送
根据产品板块和行业分析相关产品，向用户推荐类似产品
